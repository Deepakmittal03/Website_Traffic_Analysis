# 🚦 Traffic Data Analysis

## 📌 Project Overview

The **Traffic Data Analysis** project focuses on analyzing user interaction and traffic activity data to uncover engagement patterns, geographic trends, and content performance insights. The dataset contains event-level records such as clicks, user interactions, countries, cities, artists, tracks, albums, and unique link identifiers.

The primary objective of this project is to transform raw traffic data into meaningful insights using Exploratory Data Analysis (EDA), helping stakeholders understand user behavior, content popularity, and regional engagement trends.

---
  
## 🎯 Objectives

- Analyze user interaction events and traffic patterns.
- Identify the most popular artists, albums, and tracks.
- Understand geographic distribution of traffic.
- Examine country-wise and city-wise engagement.
- Discover high-performing links and content.
- Visualize traffic trends using Python libraries.
- Generate actionable business insights from user activity data.

---

## 📂 Dataset Information

The dataset contains approximately **226,000+ records** and includes the following features:

| Column | Description |
|----------|-------------|
| Event | User interaction type (Click, Preview, etc.) |
| Date | Event date |
| Country | User country |
| City | User city |
| Artist | Artist name |
| Album | Album name |
| Track | Track name |
| ISRC | International Standard Recording Code |
| LinkID | Unique content link identifier |
